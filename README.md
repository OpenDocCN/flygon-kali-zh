## 下载

### Docker

```
docker pull apachecn0/freelearn-kali-zh
docker run -tid -p <port>:80 apachecn0/freelearn-kali-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-kali-zh
freelearn-kali-zh <port>
# 访问 http://localhost:{port} 查看文档
```
